## üìñ Overview

`markview.nvim` is a tree-sitter based **markdown** previewer for Neovim. It supports additional language that are typically used alongside markdown.

The main goal of the plugin is to be *fast* while not compromising it's *looks*. It was originally designed to be used inside `Termux`.
