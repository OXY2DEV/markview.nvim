The plugin comes with a single command, `:Markview`. It can be used trigger
any of the following sub-commands,

>vim

  â€¢ `Toggle` Toggles the preview on all attached buffers! Buffers that have

  â€¢ `Enable` Enables preview on all attached buffers.

    provided).

  â€¢ `Disable<bufnr>` Disables preview on the given buffer(current buffer if
    not provided).
